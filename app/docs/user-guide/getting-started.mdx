# Getting started

Ascendio is a focused job application tracker that keeps every application, interview, and offer in one calm, organized place. This guide walks you through signing in and adding your first job.

## Prerequisites

You only need one thing to use Ascendio:

- A **Google account** — Ascendio uses Google OAuth for authentication. No password, no email verification, no forms.
